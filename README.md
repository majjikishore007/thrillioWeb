# Tech-stack:

## This project is designed based on MVC architecture

     java servlets and jsp,
     Html ,css ,Sass
     MySql

## Features

- user login
- Seaching avilaible books for bookmarking
- Bookmarking a certain book to wishList
- Remove Bookmarks from wishList
